# USB-C HighSpeed Test Fixture

This is a simple test fixture for testing Highspeed (USB 2.0) on USB-C Connectors

## How To Build

This board is intended to be manufactured on 2 Layers with 0.8 mm prepreg.

First prototypes are made with OSHPARK "2oz-0.8mm" process

Needed Parts:

1 pcs. Wuerth Elektronik 632712000011 (USB-C Connector)
2 pcs. Rosenberger 32K243-40ML5 (SMA Connector)
